# ColorShop
- 원하는 밑그림에 AI를 활용하여 색을 입히는 사이트
- 사진 편집 프로그램인 포토샵에서 영감을 따 온 이름으로 원하는 이미지에 ‘색을 입힌다’는 사이트의 목적에 따라 컬러샵이라는 이름을 짓게 되었다.

# 핵심 기능
- 인기 포스트 나열(BEST 8)
- 트위터 타임라인
- 로그인, 회원가입
- 딥러닝으로 이미지 변환
- 결과 이미지 파일 데이터베이스에 저장 후 게시글 작성
- 게시글 수정, 삭제, 좋아요 기능
- 댓글 작성, 수정, 삭제
- 이미지 게시글 리스트를 좋아요순/최신순으로 정렬
- 회원정보 보기, 수정, 탈퇴
- 내가 쓴 게시글, 좋아요한 게시글 보기
